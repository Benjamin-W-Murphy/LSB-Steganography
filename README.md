# LSB-Steganography
支持二值图隐写、小尺寸彩色图隐写、文本隐写

2023.4.17 更新了模拟攻击代码

2023.4.25 项目功能测试完成，新增灰度图隐写、攻击模拟、隐写分析

文件名含义：

- LSB-main.py    LSB隐写主函数
- RGB_image.py    RGB图隐写
- gray_image.py  灰度图隐写
- binary_image.py    二值图隐写    
- txt.py    文本隐写
- decryption.py    加密
- encryption.py    解密
- attack.py    攻击模拟
- scramble.py    混沌Logistics映射
- steganalysis.py    隐写分析
